Dynamic Content

Tableau Modal

I wanted to bring my CA1 tableau dashboard into the project and on the portfolio page if you click the tableau logo it will open a modal that links to tableau online which I have given public access. Unfortunately anyone who wants to view it needs to have a tableau online account as they don't allow anyone to see the dashboard without logging in.

Custom 404 page

In class we covered how important it is to give the customer feedback when they encounter an error and for that reason I created a custom 404 page by getting a javascript game online(reference link in Html and JS file) and implementing a 404 page. I taught I couldn't have a 404 page as my project is just html and css but I found a tutorial for github pages which allows for a 404 page by including a yaml config at the start of the page. In order to test you need to go to Github index.html if you change it to index2 it would normally give a 404 but my page give the user a 404 message and an asteroids javascript game with a nav bar to return to navigate off the page.

Footer dynamic content

I added to the footer some dynamic content firstly I used a public api called simple weather to return the weather in Dublin and post it to the page, I then used javascript to put an autoupdate time script on the page to put a proper clock that isn't static and lastly I went to free time and date to add the current date in an iframe instead of just getting it from local javascript file. I also added a visit counter from www.freecounterstat.com

I considered usability for the website and all pages a simple minimlistic layout that should have a very low learning curve with the tableau modal being the only exception but that was out of my control as tableau control who can use their services and while I considered removing the modal I think it adds to the website.

Common Usability Mistakes

I have tried to avoid the 9 common mistakes covered in class with pagenation being used for projects on my skills page and not having tiny clickable areas, I have given all pages their own titles and I have included a contact modal in the footer for all pages with a search in the nav bar that is wired up to google search.

Gestalt Laws Used
